Tipos de Triângulo:

Equilátero
Todos os lados iguais
Isósceles
Dois lados iguais
Escaleno
Todos os lados diferentes
Aguardando verificação
Digite os comprimentos dos três lados

Regras de Classificação:

  • Equilátero: Todos os três lados são iguais (L1 = L2 = L3)
  • Isósceles: Dois lados são iguais (L1 = L2 ou L1 = L3 ou L2 = L3)
  • Escaleno: Todos os lados são diferentes (L1 ≠ L2 ≠ L3)
